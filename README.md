ihover-rails
===================


Installation
=======

Add this extension to your Gemfile:

```ruby
gem "ihover-rails"
```

Then run:

```
bundle install
```

Include the application.css.sass or application.css.scss (Ruby on Rails):

```
@import ihover
```

Or application.css:

```
*= require ihover
```
